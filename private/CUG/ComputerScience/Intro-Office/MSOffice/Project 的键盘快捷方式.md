# Project 的键盘快捷方式				 

许多用户发现，将外部键盘与键盘快捷方式配合使用， Project 可帮助他们更高效地工作。 对于行动不便或者有视力障碍的用户，键盘快捷方式比触摸屏更方便，是使用鼠标的重要替代方式。

**注意:** 

- 本主题中的这些快捷方式指的是美式键盘布局。 其他键盘布局的键可能与美式键盘上的键不完全对应。
- 一个快捷方式中的加号 (+) 意味着你需要同时按多个键。
- 以逗号标记 (，) 在快捷方式中，你需要按顺序按多个键。
