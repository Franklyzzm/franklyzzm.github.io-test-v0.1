nano 快捷鍵      

编辑器的按键及其功能         

Overview of nano's shortcuts      

The editor's keystrokes and their functions         


文件处理 File handling
Ctrl+S   保存当前文件
Save current file
Ctrl+O提供写入文件（"另存为"）
Offer to write file ("Save as")
Ctrl+R将文件插入到当前文件中
Insert a file into current one
Ctrl+X关闭缓冲区，从nano退出
Close buffer, exit from nano

編輯 Editing
Ctrl+K   将当前行切割到切割缓冲区中
Cut current line into cutbuffer
Alt+6将当前行复制到cutbuffer中
Copy current line into cutbuffer
Ctrl+U粘贴剪切缓冲区的内容
Paste contents of cutbuffer
Alt+T剪切直到缓冲区结束
Cut until end of buffer
Ctrl+]完成当前单词
Complete current word
Alt+3註釋/非註釋 行/区域
Comment/uncomment line/region
Alt+U撤销最后一个动作
Undo last action
Alt+E重做最后一个未完成的动作
Redo last undone action

查找與替換 Search and replace
Ctrl+Q   开始向後搜索
Start backward search
Ctrl+W开始向前搜索
Start forward search
Alt+Q向后查找下一次出现的情况
Find next occurrence backward
Alt+W向前查找下一次出现的情况
Find next occurrence forward
Alt+R开始替换会话
Start a replacing session

刪減 Deletion
Ctrl+H删除光标前的字符
Delete character before cursor  
Ctrl+D删除光标下的字符
Delete character under cursor
Ctrl+Shift+Del  删除左边的单词
Delete word to the left
Ctrl+Del删除右边的单词
Delete word to the right
Alt+Del删除当前行
Delete current line

操作 Operations
Ctrl+T   运行拼写检查
Run a spell check
Ctrl+J调整段落或区域
Justify paragraph or region
Alt+J调整整个缓冲区
Justify entire buffer
Alt+B运行语法检查
Run a syntax check
Alt+F运行一个格式化器/固定器/排列器
Run a formatter/fixer/arranger
Alt+:开始/停止宏的记录
Start/stop recording of macro
Alt+;重放宏
Replay macro



 移動 Moving around
Ctrl+B   后退一个字符
One character backward
Ctrl+F前置一个字符
One character forward
Ctrl+←一个單詞倒退
One word backward
Ctrl+→一个單詞前进
One word forward
Ctrl+A到行的开始
To start of line
Ctrl+E至行末
To end of line
Ctrl+P一行向上
One line up
Ctrl+N一行向下
One line down
Ctrl+↑到上一个区块
To previous block
Ctrl+↓到下一个区块
To next block
Ctrl+Y上一页
One page up
Ctrl+V下一页
One page down
Alt+\至缓冲区顶部
To top of buffer
Alt+/至缓冲区末端
To end of buffer

特別行動 Special movement
Alt+G   转到指定行
Go to specified line
Alt+]转到补充括号
Go to complementary bracket
Alt+↑向上滚动视口
Scroll viewport up
Alt+↓向下滚动视口
Scroll viewport down
Alt+<切换到前面的缓冲区
Switch to preceding buffer
Alt+>切换到后续缓冲区
Switch to succeeding buffer

信息 Information
Ctrl+C   报告光标位置
Report cursor position
Alt+D报告字数/行数/字符数
Report word/line/char count
Ctrl+G显示帮助文字
Display help text

其他 Various
Alt+A打开/关闭标记
Turn the mark on/off
Tab缩进标记区域
Indent marked region
Shift+Tab   取消标记区域
Unindent marked region
Alt+N打开/关闭行号
Turn line numbers on/off
Alt+P打开/关闭可见的空白区域
Turn visible whitespace on/off
Alt+V逐字逐句输入下一个键程
Enter next keystroke verbatim
Ctrl+L刷新屏幕
Refresh the screen
Ctrl+Z暫停 nano
Suspend nano
