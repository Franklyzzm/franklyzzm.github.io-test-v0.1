# 操作系统 hosts 文件

## Windows

hosts 文件位置：

` C:\Windows\System32\drivers\etc\hosts`

修改：`hosts`

## Mac

hosts 文件位置：

`/etc/hosts`

修改：

```
sudo vi /etc/hosts
sudo vim /etc/hosts
sudo nano /etc/hosts
```



## Linux

hosts 文件位置：

`/etc/hosts`

修改：

```
sudo vi /etc/hosts
sudo vim /etc/hosts
sudo nano /etc/hosts
```
