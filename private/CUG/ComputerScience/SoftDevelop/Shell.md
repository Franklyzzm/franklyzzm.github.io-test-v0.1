A ZSH framework like oh-my-zsh
Installing
After installing oh-my-zsh

Download the theme here

Put the file agnosterzak.zsh-theme in $ZSH_CUSTOM/themes/

Configure the theme in your ~/.zshrc file:

ZSH_THEME="agnosterzak"
